# 🏗️ BuildTrack — Construction Site Manager

A full-featured construction site management app built with React + Vite.

## Modules

| Module | Description |
|--------|-------------|
| 📊 Dashboard | Live stats, phase progress, budget overview, stock alerts |
| ✅ Tasks | Kanban cards with progress tracking, phases & priorities |
| 👷 Workers | Attendance heatmaps, wage calculator, 30-day tracking |
| 📦 Materials | Inventory with auto stock status & supplier tracking |
| 💰 Budget | Transactions, category breakdown, income vs expenses |
| 🤖 AI Insights | Claude AI — ask anything about your site |

## Getting Started

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

Push to `main` — GitHub Actions auto-builds and deploys.

## Tech Stack

- React 18 + Vite 5
- Claude AI (claude-sonnet-4-20250514)
- IBM Plex Sans + Syne fonts
- No external UI libraries
